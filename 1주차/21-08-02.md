## AI Math 강의 : 1 ~ 2강

- 벡터가 뭐에요?
```
import numpy as np
x = np.array([1,7,2])
y = np.array([5,2,1])

# 성분곱(Hadamard product)
x * y


# L1 노름 : 각 성분의 변화량의 절대값
def l1_norm(x):
  x_norm = np.abs(x)
  x_norm = np.sum(x_norm)
  return x_norm

# L2 노름 : 유클리드 거리
def l2_norm(x):
  x_norm = x*x
  x_norm = np.sum(x_norm)
  x_norm = np.sqrt(x_norm)
  return x_norm
```
