# 깃허브 특강

새로 배운 것
- origin : 원격 저장소
- origin/main : 원격 저장소의 최신 버전
- main == master : 로컬 저장소의 최신 버전
- 이전 버전에서 최신 버전으로 돌아올 때는 master 아이콘에 checkout branch를 해야 한다.
- 브랜치 앞 동그라미는 head


확장 프로그램 추천
- git graph : 소스트리처럼 그래프로 브랜치를 확인할 수 있다.
